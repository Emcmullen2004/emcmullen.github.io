Lists


Unordered

Unordered lists are, if i were to simplify it, its a method to make bullet points. It is defined as "ul".
When making any list you first have to define that its a list and what that list is, after that you can start to create the list items. Ex:
<ul>
  make sure to indent<li>place list item here</li>
</ul>

  • place list item here

Ordered

As the name implies, unlike the unordered list it has an order to it. Its defined with "ol".
By default its ordered with numbers, however you can change that using the "type" atribute.
"a" will make it lower case letters from a - z. "A" will be upper case letters, A - Z.
"i" will make it lowercase romain numerals, i, ii, iii... Likewise "I" will be upper case romain numerals. Ex:
<ol you can place a type atribute here>
  <li>write first list item here<li>
</ol>>

Description

If I were to describe a description list it would be an easier way to make a unordered list with one subcatigory.
As the name implies, a description list allows you to name an item and give a brief description. It's defined as:
Description list "dl", description term "dt", description X2 "dd". Ex:
<dl>
  <dt>item goes here</dt>
    <dd>place descrpiton here</dd> </dl>
