HTML Components: Part 1


First we did a recap on HTML Basics

Next we learned about images. The image tag places images into your website. It is written as:
<img src="write link here" alt="write alternative text here">
The image tag is the first self closing tag we learned about, self closing means that you dont need to add a copy of that tag with a "/" in it.
Also instead of a link you can use the file name saved on your computer.

Next we learned about inline and block elements, and their differences. Block elements start and end with a line break. some examples are headings and paragraphs.
Inline elements are all others, aka. elements that dont start or end with a line break. some examples are images and links. They only take up as much space as they need to.
To add a line break manually you just type <br/>, this is another example of a self closing tag.

Now we have developer tools. It's a way to access the code from other webites in real time. To use them you use the "fn" key and press "f12", or press "control+shift+i".
You can also use the three dots at the top right and then click developer tools in the drop-down window.
If you use the selector tool you can go over the elements in the webpage, when you click them you get shown where it is in the dev. tools menu.
